二叉树受虐经验1

发表于 2020-04-13 | 分类于 算法

题目：层数最深的叶子节点的和

题源leetcode

原本的想法：先递归求出最大深度，然后在另一个递归中把最深的叶子累加。因为一开始想不出怎么在一个递归中解决深度和累加的问题。。其实完全在一个前序遍历框架就完成了，

class Solution {
public:

int maxDepth;
int sum=0;
int getMaxDepth(TreeNode* root){
    if(root==NULL) return 0;
    return max(getMaxDepth(root->left),getMaxDepth(root->right))+1;
}

void getSum(TreeNode* root,int depth){
    if(root==NULL) return ;
    if(depth==maxDepth) sum+=root.val;
    else{
        getSum(root->left,depth+1);
        getSum(root->right,depth+1);
    }
}
int deepestLeavesSum(TreeNode* root) {
    maxDepth = getMaxDepth(root);
    getSum(root,1);
    return sum;
}


};

好的做法是一个框架就能搞定，关键是要有一个可维护的maxdepth变量。你每次遍历到新的一层，depth就会加一，那maxdepth就可能需要更新了。更新条件就是depth>maxdepth,更新完maxdepth后，一定要初始化total为当前节点的值。因为total真正的初始值就是第一个遍历到的最深节点，而每一个能更新maxdepth的节点都有可能就是第一个遍历到的最深节点。因此total的初始值也要随之更新。

class Solution {
private:
int maxdep = -1;
int total = 0;

public:
void dfs(TreeNode* node, int dep) {
    if (!node) {
        return;
    }
    if (dep > maxdep) {
        maxdep = dep;
        total = node->val;  //关键点：你每次更新最大深度，就要初始化一次total,因为该节点有可能就是第一个遍历到的最大深度的叶子结点
    }
    else if (dep == maxdep) {
        total += node->val;
    }
    dfs(node->left, dep + 1);
    dfs(node->right, dep + 1);
}

int deepestLeavesSum(TreeNode* root) {
    dfs(root, 0);
    return total;
}
};

总结：要善于将辅助变量放在参数或函数体内使用

求斐波那契数列

发表于 2020-03-23

题目

输入 n ，求斐波那契（Fibonacci）数列的第 n 项。斐波那契数列的定义如下：
F(0) = 0, F(1) = 1
F(N) = F(N - 1) + F(N - 2), 其中 N > 1.
斐波那契数列由 0 和 1 开始，之后的斐波那契数就是由之前的两数相加而得出。

分析

法一：迭代；法二：递归；这两个都比较普通，无法解决时间超时等问题。
法三：公式法，这个还没怎么用过，应为总怕精度会出现问题Orz。法四：退出矩阵递推方程，用矩阵快速幂完美解决。下面就主讲法四。

如何推出矩阵递推方程

先看看这个式子F(N) = F(N - 1) + F(N - 2)，明显就是后一项由前两项递推得出。但这种递推不是很对称哈哈。我们比较喜欢的是后一项由前一项推出。那有啥方法可以变换一下，肯定是有的。写成[F(N),F(N - 1)] = [F(N-1),F(N - 2)] * [[a,b],[c,d]]。再看看，对称了！你令Xn = [F(N),F(N - 1)]，那Xn-1不就是[F(N-1),F(N - 2)]了吗？好，那为啥要乘个[[a,b],[c,d]]，你想啊，这1×2的矩阵不就等于1×2的矩阵乘以2×2的矩阵吗？因此我们确定一下这个[[a,b],[c,d]]吧。好要注意到F(N) =F(N - 1) * 1 + F(N - 2) * 1 ; F(N - 1) = F(N - 1) * 1 + F(N - 2) * 0。 wow，系数矩阵这不就出来了吗，[[1,1],[1,0]]

递推方程：(f(n),f(n-1))=(f(n-1),f(n-2)) * [[1,1],[1,0]]

不难发现，最后写下去应该是(f(n),f(n-1))=(f(1),f(0)) * [[1,1],[1,0]] ^ n-1。 好，那么问题就仅剩如何解决[[1,1],[1,0]] ^ n-1啦！

矩阵的快速幂

好，不就是[[1,1],[1,0]] ^ n-1吗，老子乘个n次，但有些题就是连O(n)都给你卡。哦，那就要想O(logn)了，那么问题来了，如何降到O(logn)呢？想一下，这个幂次n-1真的要一次一次弄吗？要知道任何数都可以拆成2的n次方相加的呀，比如7 = 2^2 + 2 + 1。 那7次就降到3次了。

两个矩阵的普通相乘

Matrix mul(Matrix A, Matrix B)
{
Matrix temp;
temp.len1 = A.len1;
temp.len2 = B.len2;
//三层循环搞矩阵相乘，公式temp.mat[i][j] += A.mat[i][k] * B.mat[k][j]
for(int i=0;i<temp.len1;i++)
    for(int j=0;j<temp.len2;j++)
    {
        temp.mat[i][j] = 0;
        for(int k=0;k<A.len2;k++){
            temp.mat[i][j] = (temp.mat[i][j] + ((A.mat[i][k]%mod) * (B.mat[k][j]%mod))%mod)%mod;
        }
    }
return temp;
}

矩阵的快速幂核心

Matrix matPow(Matrix base, Matrix pow_mat, int num)  //base是基矩阵，pow_mat是幂次矩阵，最后基矩阵将变成结果矩阵
{
while(num){
    if(num&1)   //二进制最后一个数为1, 则base要乘一个pow_mat
    {
        base = mul(base,pow_mat);
    }
    pow_mat = mul(pow_mat,pow_mat); //pow_mat要翻倍
    num = num>>1;   //num除以2
}
return base;
}
