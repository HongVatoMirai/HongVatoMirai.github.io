git的基本使用和管理

发表于 2020-03-22 | 分类于 git

前言

 第一次真正意义上去学习使用git和github，为了纪念这It界的盛事，我决定写篇文章说说使用git的基本命令 //现在已经是老菜鸟了Orz,再不学习就凉了。

git管理本地仓库

git初始化及仓库创建和操作

基本信息设置
设置用户名
git config –global user.name ‘yourname’
设置用户名邮箱
git config –global user.email ‘youremail’

向仓库添加文件

  • Git add (工作目录->暂存区)
  • Git status 查看提交状态
  • Git commit -m “”(暂存区->仓库)

删除仓库文件

  • Rm -rf xxx 删除文件
  • Git rm xxx 从git中删除文件
  • Git commit -m “”

Git 管理远程仓库

  1. Git clone ‘仓库地址’ #先克隆下来
  2. 做自己的修改
  3. git add
  4. git commit -m
  5. git push

结语

 只是一些很基本的操作，github上的fetch和pull，分支和合并等等之后再谈吧。。
